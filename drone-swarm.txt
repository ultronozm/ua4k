DESCRIPTION Core rule: both drones move on every command. Park drone "1" on "a" and drone "2" on "b". Parked drones ("A"/"B") lock in place. If one drone is blocked, it holds position while the other can keep moving.

DESCRIPTION New mechanic: stepping onto key "k" triggers an unlock side effect that opens all gates "g". This map is designed so the unlock timing and approach angle both matter.

DESCRIPTION Final challenge: multiple gates and long detours. Use lock, release ("z"), and key timing to keep both drones on viable paths.

BIND w north s south a west d east z release

CMD north
 ATOMIC
  CALL_EACH move1_north move2_north

CMD south
 ATOMIC
  CALL_EACH move1_south move2_south

CMD west
 ATOMIC
  CALL_EACH move1_west move2_west

CMD east
 ATOMIC
  CALL_EACH move1_east move2_east

CMD release
 TRY_ALL
  A 1

  B 2

CMD unlock
 TRY_ALL
  g -

CMD move1_north
 MATCH1
  A A

  FOR s xg2BbA
   s s
   1 1

  a A
  1 -

  k 1 unlock
  1 -

  - 1
  1 -

CMD move1_south
 MATCH1
  A A

  FOR s xg2BbA
   1 1
   s s

  1 -
  a A

  1 -
  k 1 unlock

  1 -
  - 1

CMD move1_west
 MATCH1
  A A

  FOR s xg2BbA
   s1 s1

  a1 A-

  k1 1- unlock

  -1 1-

CMD move1_east
 MATCH1
  A A

  FOR s xg2BbA
   1s 1s

  1a -A

  1k -1 unlock

  1- -1

CMD move2_north
 MATCH1
  B B

  FOR s xg1AaB
   s s
   2 2

  b B
  2 -

  k 2 unlock
  2 -

  - 2
  2 -

CMD move2_south
 MATCH1
  B B

  FOR s xg1AaB
   2 2
   s s

  b B

  2 -
  k 2 unlock

  2 -
  - 2

   s2 s2

  b2 B-

  k2 2- unlock

  -2 2-

   2s 2s
